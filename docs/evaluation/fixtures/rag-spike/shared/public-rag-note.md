# Public RAG Evaluation Note

This public guidance note is visible to every user.

RAG evaluation should not rely on Recall@5 alone. A useful report should also check faithfulness, citation coverage, source correctness, forbidden hit count, and whether the answer admits insufficient evidence when retrieval is weak.

The note is intentionally public so both user A and user B can retrieve it.
