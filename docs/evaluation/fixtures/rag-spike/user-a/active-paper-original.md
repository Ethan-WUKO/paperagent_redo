# Delta-Graph-RAG Original Manuscript

This original manuscript describes Delta-Graph-RAG, a retrieval method that connects citation graph neighborhoods with dense passage retrieval.

The original evaluation reported Recall@5 0.71 on the Yanban scholar note benchmark. Citation coverage was incomplete because the first draft used manual bridge edges between related literature cards.

This original version is superseded by the polished manuscript. It should be used only when the user explicitly asks for the original draft.
