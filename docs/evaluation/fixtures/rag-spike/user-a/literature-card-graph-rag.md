# Literature Card: GraphRAG with Edge-Conditioned Retrieval

Title: Edge-conditioned retrieval for GraphRAG systems.

Authors: Lin Qiao, Marta Iversen, Daniel Koh.

Year: 2025.

Venue: Journal of Retrieval-Augmented Systems.

DOI: 10.5555/graph-rag.2025.

Summary: The paper argues that GraphRAG benefits from edge-conditioned retrieval when graph links represent citation support instead of generic similarity. It recommends preserving DOI, venue, year, and source URL metadata when a literature card is injected into a generation prompt.

Relevance to user A: This card supports the Delta-Graph-RAG section on learned bridge-edge ranking.
