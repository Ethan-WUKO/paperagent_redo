# 研伴 ScholarAI 项目演示文档

研伴 ScholarAI 是一个面向研究生和科研工作者的私有研究助手。项目目标是把日常科研中的知识库问答、文献搜索、论文润色和多步骤 Agent 工作流放到同一个工作台里。

当前演示重点包括：

- 私有知识库：用户可以上传 Markdown、TXT、PDF、DOCX 等资料，系统解析文本、切块、生成向量，并支持 RAG 检索。
- 聊天助手：聊天页支持流式输出，能结合用户知识库、网页搜索和工具调用回答问题。
- Agent 计划模式：复杂任务可以先拆成步骤，再执行、观察工具结果，并持续更新任务状态。
- 论文处理：Paper Polish 工作区支持上传 LaTeX 主文件和 bib 文件，生成论文修改建议、文献证据和导出产物。

项目的核心价值不是替代研究者，而是把分散在文件、网页、笔记和论文里的信息组织起来，让研究者更快定位依据、制定计划和推进写作。
